Gas! Notify Me

เครื่องตรวจจับควันอัจฉริยะ

บทคัดย่อ

  เนื่องจากอัคคีภัย(หรือไฟไหม้)เหตุการณ์อันตรายอันเกิดจากไฟที่ขาดการควบคุมดูแล ทำให้เกิดการติดต่อลุกลามไปตามบริเวณที่มีเชื้อเพลิงเกิดการลุกไหม้ต่อเนื่อง สภาวะของไฟจะรุนแรงมากขึ้นถ้าการลุกไหม้ที่มีเชื้อเพลิงหนุนเนื่องหรือมีไอของเชื้อเพลิงถูกขับออกมามากความร้อนแรงก็จะมากยิ่งขึ้น สร้างความสูญเสียให้ทรัพย์สินและชีวิต ทั้งนี้การที่มีการป้องกันดีหรือการรับรู้ภัยที่กำลังเกิดขึ้นได้ทันท้วงที่ก็เป็นสิ่งที่จำเป็นอย่างมากจึงได้เกิน ทางกลุ่มของเราได้ทราบถึงปัญหานี้จึงได้สร้างอุปกรณ์ “ดักจับควัน” เพื่อวัตถุประสงค์ที่จะป้องกันการเกิดอัคคีภัยโดยการยับยั้งได้ทันท้วงที โดยจะมีขั้นตอนดังนี้ เมื่อมีควันเกิดขึ้นและเข้าใกล้ตัวของอุปกรณ์ดักจับควัน อุปกรณ์ดักจับควันจะทำงานทันทีโดนการส่งข้อความแจ้งเตือนการเกิดอัคคีภัยไปยังแอปพลิเคชันLine และ ยังมีไฟแจ้งเตือน มีเสียงออดแจ้งเตือน ณ ตัวอุปกรณ์ด้วย
ผลการศึกษาค้นคว้า การทดลองการใช้งานของอุปกรณ์ดักจับควัน โดยใช้ควันไฟจากไฟเเช็ก ผลการทดลองการทำงานที่ได้อุปกรณ์ดักจับควันมีการทำงานที่แม่นยำ100% เมื่อมีควันที่เข้าใกล้จะมีการแจ้งเตือนข้อความไปยังแอปพลิเคชันLine และมีไฟแจ้งเตือน มีเสียงออดแจ้งเตือนทุกครั้งที่เกิดขึ้น

วัสดุและอุปกรณ์

1. NodeMCU (ESP8266)
2. บอร์ดสำหรับพัฒนา (Breadboard)
3. สายไฟ(ผู้-เมีย)
4. เซนเซอร์ตรวจจับแก๊ส MQ-2
5. ออด(Piezo)
6. หลอดไฟ LED 2 สี (เขียว-แดง)

วิธีการทำงาน

เมื่อเริ่มรันโปรแกรม NodeMCU_LINE.ino ไฟแจ้งเตือนจะเป็นสีเขียว เมื่อเราจุดแก๊ส หรือจุดควันไฟ
เซนเซอร์ MQ-2 จะตรวจจับ และเปลี่ยนไฟแจ้งเตือนเป็นสีแดง จากนั้นออดจะดังขึ้น และโปรแกรมจะต่อ Wi-Fi ผ่าน NodeMCU
เพื่อแจ้งเตือนว่าพบแก๊สและค่าของเซนเซอร์ออกไปที่ Line Notify

ผลการทดลอง

ผลการทดลองการทำงานที่ได้อุปกรณ์ดักจับควันมีการทำงานที่แม่นยำ เมื่อมีควันที่เข้าใกล้จะมีการแจ้งเตือนข้อความปริมานควันไปยังแอปพลิเคชันLine และมีไฟแจ้งเตือนบริเวณบอร์ด มีเสียงออดแจ้งเตือนทุกครั้งที่เกิดขึ้น เพื่อให้เตรียมตัวรับมือกับอัคคีภัยที่อาจจะเกิดขึ้นได้ทันทวงที
